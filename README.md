# TangoRikai---Japanese-Vocab-Chrome-Extension

# Introduction
単語理解 (Tango Rikai) is a chrome extension that helps users find and understand vocabulary. 
With this extension, you can:
 * See homophones and synonyms of any word
 * learn the kanji of any word
 * See sentences for any word
   
To use:
 * Highlight any Japanese word
 * press the 検索 button above the word or use the shortcut Ctrl+Shift+S
 * Click on the respective Homophone, Synonym or Kanji Tab to see words
 * Click on a specific card to learn more
 * Use the buttons on the selected word to hear the pronounciation, see sentences, or add the word to your Bunpro reviews

# Download Instructions
 * Download and unzip the files from this repository
 * Go to "chrome://extensions/" and turn on Developer Mode
 * Click on "load unpacked" and select the file
 * Use the extension!
